# 🏥 Health Insurance Analytics — Data Science Project

## 📌 Visão Geral
Projeto de Ciência de Dados focado no setor de Plano de Saúde, com objetivo de gerar insights estratégicos sobre custos, churn e perfil de clientes.

Este projeto é voltado 100% para portfólio, simulando um cenário real de uma operadora de saúde.

---

## 🎯 Objetivos de Negócio
- Reduzir churn de clientes
- Identificar padrões de alto custo (sinistralidade)
- Melhorar rentabilidade dos planos
- Apoiar decisões estratégicas

---

## 📊 Principais Análises

### 📉 Sinistralidade
Avaliação da relação entre custo médico e receita.

### 🔄 Churn
Identificação de clientes com maior probabilidade de cancelamento.

### 👥 Perfil de Clientes
Distribuição por idade, plano e comportamento.

### 💰 Financeiro
Análise de receita, custo e margem.

---

## 💡 Insights Estratégicos
- Clientes acima de 60 anos possuem maior custo médico
- Planos básicos concentram maior churn
- Pequena parcela dos clientes gera maior custo
- Planos premium apresentam maior retenção
